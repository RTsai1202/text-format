# 還是忍不住花一小時多
把 YouTube 影片轉錄功能做出來了
使用網友推薦的 Groq Whisper（真的快！）
然後它可以判斷：

有字幕 → 優先取用，不轉錄
無字幕 → 通常是中文，轉錄

並且直接存到指定資料夾
用 Gemini CLI 進行摘要與追問（免費）
之後還可以加入批次轉錄與摘要的功能
終於可以來清一清我的 YouTube 待播清單了
